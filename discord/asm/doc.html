Assembling code

@Assembler <architecture> <code block> Assembles code. Architecture can be one of:
i386, x86 for Intel-style x86 assembly
x64, x86_64 for Intel-style x64 assembly
i386_att, x86_att for AT&T-style x86 assembly
x64_att, x86_64_att for AT&T-style x64 assembly
armv6, armv6k for ARMv6 assembly
armv7, armv7a for ARMv7 assembly
aarch64, armv8 for ARMv8 assembly
